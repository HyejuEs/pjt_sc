# 변경 요약 (4차 수정)

## 표 밖 중복 텍스트 - "제거"에서 "표시만"으로 전면 변경 (핵심)
- ★ WER은 중복을 삭제/수정하지 않고 그대로 포함해 계산한다.
- report.html에서만 표 밖에 중복 추출된 원문 라인을 파란색(dup-line) + "표 내용 중복"
  뱃지로 표시한다. 별도 "3. 표 내용 중복 추출" 섹션에도 목록으로 정리(참고용).
- 판정: 표 셀값의 약 90%+ 가 표 밖 본문에 통째로 중복 추출된 라인만 표시. 한두 셀만
  우연히 겹치면 표시 안 함. 완전히 빈 표는 판정 제외. 표(TEDS)/글 사이 중복은 안 건드림.
- 함수명: remove_table_dup_text → detect_table_dup_text (삭제 안 하고 감지만).

## 표 밖 중복 테스트 케이스 (ppt_table_dup) - before/after 중립
- before/after 둘 다 표(HTML) 값이 표 밖 본문에도 통째로 중복 추출된 상황으로 구성.
- WER은 중복 포함해 계산되어 점수가 나빠지고, 리포트에는 그 중복 라인이 파란색으로 표시됨.

## (이전 유지) 그 외
- PPT 제목: 슬라이드 내 최대 폰트 크기 문단 = 제목 → "## "(볼드 제거). 크기 정보 없으면
  title placeholder 폴백. 일반 텍스트 볼드는 "**..**" 유지. ## 전처리는 GT 추출에서만.
- 슬라이드 단위 WER: 슬라이드를 원소로 Needleman-Wunsch 정렬 후 쌍마다 WER(by_unit 확인).
- report.html: 슬라이드/섹션 구분선(━━), "(해당 없음)" 공백 정렬, 줄바꿈 유연화 내역 표시.
- 줄바꿈 유연화: 인접 ±1줄만 확인해 GT 경계에 맞춤(합침/나눔). 밀림 방지.
- 듀얼 GT: before.txt↔before_gt, after.txt↔after_gt. 리포트에 두 GT 각각 2열 표시.
- xlsx: 시트명 ## 추출 + 시트명 텍스트 WER 병행. 표는 TEDS.
